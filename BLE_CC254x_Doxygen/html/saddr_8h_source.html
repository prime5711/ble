94 /****************************************************************************
95  * @fn sAddrIden
96  *
97  * @brief Check if two device addresses are identical.
98  *
99  * This routine is virtually the same as sAddrCmp, which is used
100  * to determine if two different addresses are the same. However,
101  * this routine can be used to determine if an address is the
102  * same as a previously stored address. The key difference is in
103  * the former case, if the address mode is "none", then the
104  * assumption is that the two addresses can not be the same. But
105  * in the latter case, the address mode itself is being compared.
106  * So two addresses can be identical even if the address mode is
107  * "none", as long as the address mode of both addresses being
108  * compared is the "none".
109  *
110  * input parameters
111  *
112  * @param pAddr1 - Pointer to first address.
113  * @param pAddr2 - Pointer to second address.
114  *
115  * output parameters
116  *
117  * @return TRUE if addresses are identical, FALSE otherwise
118  */
119 extern bool sAddrIden(const sAddr_t *pAddr1, const sAddr_t *pAddr2);
